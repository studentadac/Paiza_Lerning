自然数 A, B, C が与えられます。(A, B, C の最大値) - (A, B, C の最小値)を答えてください。

入力される値
A B C

入力値最終行の末尾に改行が１つ入ります。
文字列は標準入力から渡されます。 標準入力からの値取得方法はこちらをご確認ください
期待する出力
答えを1行に出力してください。

末尾に改行を入れ、余計な文字、空行を含んではいけません。

条件
すべてのテストケースにおいて、以下の条件をみたします。

・ 1 ≦ A, B, C ≦ 100

入力例1
30 50 10

出力例1
40

入力例2
1 1 1

出力例2
0

A:
最低値minを範囲の上限100、最大値maxを0としminより小さな数字があったらminを書き換え、maxもminと逆の処理をする
最後にmax-minを出力する
